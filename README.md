# BayesianLearning

## Setup

* Install [Julia Language](https://julialang.org/) and Jupyter notebook (Anaconda3)
* Install related packages
    ```julia
    using Pkg
    Pkg.add("IJulia.jl")
    Pkg.add("Distributions.jl")
    Pkg.add("Optim.jl")
    Pkg.add("PyCall.jl")
    Pkg.add("PyPlot.jl")
    ```
* Run Jupyter notebook
