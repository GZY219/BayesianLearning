# Bayesian Learning

## Setup

* Install [Julia Language](https://julialang.org/) and Jupyter Notebook ([Anaconda3](https://www.anaconda.com/download/#macos))
* Install related packages in Julia REPL

    ```julia
    using Pkg

    Pkg.add("IJulia.jl")
    Pkg.add("Distributions.jl")
    Pkg.add("Optim.jl")
    Pkg.add("PyCall.jl")
    Pkg.add("PyPlot.jl")
    ```

* Run Jupyter notebook

    ```shell
    $ jupyter notebook
    ```
